## ğŸš€ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- **Flutter SDK** (version 3.0 or higher) - [Install Flutter](https://flutter.dev/docs/get-started/install)
- **Dart SDK** (included with Flutter)
- **Android Studio** or **Xcode** (for mobile development)
- **Git** - For version control

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/rly09/mentalhealthapp_ui.git
cd mentalhealthapp_ui
```

2. **Get Flutter dependencies**

```bash
flutter pub get
```

3. **Run the app**

For Android:
```bash
flutter run -d android
```

For iOS:
```bash
flutter run -d ios
```

For Web:
```bash
flutter run -d chrome
```

For other platforms (Windows, macOS, Linux):
```bash
flutter run -d <platform>
```
